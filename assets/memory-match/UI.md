# Memory Match — UI Design Tokens

Premium casual mobile polish. Ethio Telecom palette.

## Colors

| Token | Value | Use |
|-------|-------|-----|
| `--mm-green` | `#4f9e16` | Primary actions, card backs |
| `--mm-green-light` | `#6cc52f` | Gradients, highlights |
| `--mm-blue` | `#1f74e0` | Score, links, timer |
| `--mm-blue-light` | `#4fc3f7` | Gradients |
| `--mm-gold` | `#f2b21a` | Rewards, sparkles |
| `--mm-danger` | `#e74c3c` | Mismatch flash, low timer |

## Motion

- Bounce ease: `cubic-bezier(0.34, 1.56, 0.64, 1)`
- Smooth ease: `cubic-bezier(0.4, 0, 0.2, 1)`
- Card flip: 0.45s
- Match pop: 0.5s

## Typography

- Title: 900 weight, gradient green→blue
- HUD labels: 0.62rem uppercase
- HUD values: 900 weight, tabular-nums
